# verbmeister-privacy
Privacy Policy for VerbMeister
Last updated: January 2026
This privacy policy describes how the mobile application VerbMeister (“Application”, “we”, “us”, or “our”) handles user data.
VerbMeister is an offline educational game built with the Ren’Py engine. We respect your privacy and do not collect, store, or share any personal information.
1. Information We Do Not Collect
The Application does not collect any personal data, including but not limited to:
Name, email address, phone number
Location data
Payment information
Contacts, photos, media, or files
Device identifiers
Cookies or tracking technologies
Usage analytics
The Application does not use third-party analytics, advertising, or tracking SDKs.
2. Offline Use
The Application works fully offline.
No data is sent to external servers, and no external communication takes place.
3. Third-Party Services
The Application does not integrate with any third-party services such as:
Google Analytics
Ad networks
Social networks
Game Center
Cloud services
No third-party software collects data from the Application.
4. Data Storage
Since the Application does not collect or store user data, no personal information is saved on the device or on external servers.
5. Children’s Privacy
The Application is suitable for users of all ages.
Because no data is collected, it complies with COPPA and other children’s privacy regulations.
6. Security
Although no personal data is processed, we strive to ensure the Application operates securely and safely on the device.
7. Changes to This Policy
We may update this Privacy Policy from time to time. Any changes will be posted on this page and reflected in the “Last updated” date above.
